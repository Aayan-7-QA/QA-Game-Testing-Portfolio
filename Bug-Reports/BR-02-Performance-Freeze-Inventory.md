# [Performance][Freeze] Game Freezes When 
Opening Inventory After Extended Gameplay

## Description
The game freezes for 5-10 seconds when 
the player opens the inventory after 
extended gameplay.

## Steps to Reproduce
1. Launch the game on the build 1.0.GHJI
2. Start gameplay
3. Play continuously for 20-30 minutes
4. Open the inventory

## Expected Result
Inventory should open instantly without 
any delay or freeze

## Actual Result
The game freezes for 5-10 seconds when 
opening the inventory after extended gameplay

## Details
- **Platform:** PC
- **Severity:** Medium
- **Reproducibility:** 5/5
- **Labels:** Performance, Freeze
