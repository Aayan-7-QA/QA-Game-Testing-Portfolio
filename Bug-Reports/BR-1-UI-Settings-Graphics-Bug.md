# [UI][Settings] Graphics Settings Are Not 
Applied After Clicking Apply Button

## Description
While playing a game, the player opens the 
graphic setting menu and changes high to low. 
Graphics settings do not change and the game 
continues to run on high graphics.

## Steps to Reproduce
1. Launch the game on the build 1.02.04.65796Gh
2. Start the Mission/Gameplay
3. Open the setting menu and go to the 
   graphic setting tab
4. Change the Graphic setting High to Low
5. Click to Apply

## Expected Result
Graphics settings should be applied and 
the game should switch to low graphics

## Actual Result
Graphic settings do not change from high 
to low without error messages

## Details
- **Platform:** PC
- **Severity:** High
- **Reproducibility:** 5/5
- **Labels:** Gameplay, UI
