# [Network][Connectivity] Game Behaves 
Incorrectly/Freeze on Unstable Network

## Description
While the user is playing a game on an unstable network, there is a connectivity bug 
game behaving incorrectly such as freezing or performance issues this bug leads to a bad user experience
if the user’s internet is perfectly fine

## Steps to Reproduce
1. Connect the internet into the device
2. Launch the game on the build 1.203.43
3. Start the gameplay/mission
4. Play for 5 to 10 minutes

## Expected Result
The game should run perfectly fine without any performance issues, freezes, etc
on the unstable internet connection 

## Actual Result
The game is behaving incorrectly/freezing on the unstable internet and
causing performance issues

## Details
- **Platform:** PC and PS5
- **Severity:** Medium
- **Reproducibility:** 5/5
- **Labels:** Network, Connectivity
