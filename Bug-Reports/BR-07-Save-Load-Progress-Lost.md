# [Game][Save/Load] Game Progress is Not Saved 
After Clicking the Save Button

## Description
When the player makes progress in the game and 
clicks the save button, recent progress is lost 
and not saved correctly when they exit or 
restart the game.

## Steps to Reproduce
1. Launch the game on the build 1.23
2. Start playing the game
3. Play for 5 to 10 minutes and make 
   some progress in the game
4. Click the save button and restart the game

## Expected Result
After pressing the save button, the game progress 
should be saved and stored in the database/cloud server

## Actual Result
Game progress is not saved after clicking the 
save button and all progress is reset

## Details
- **Platform:** PC
- **Severity:** High
- **Reproducibility:** 5/5
- **Labels:** Game, Save/Load
