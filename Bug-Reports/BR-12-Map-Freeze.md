# [Map][Freeze] The Game Freezes When 
it Opens the Game Map

## Description
The game freezes when the player taps on 
the map during gameplay. During the freeze, 
no error was displayed on the screen, 
prompting the player to force-close the game.

## Steps to Reproduce
1. Launch the game on the build 2.0000.276
2. Start the gameplay/mission
3. Play the game for 1 to 2 minutes
4. Tap on the map

## Expected Result
After clicking on the game map, it should 
open smoothly without any freeze or errors

## Actual Result
After clicking on the game map, the map 
became freeze in the middle of the gameplay

## Details
- **Platform:** PC
- **Severity:** Critical
- **Reproducibility:** 5/5
- **Labels:** Map, Freeze, Gameplay
