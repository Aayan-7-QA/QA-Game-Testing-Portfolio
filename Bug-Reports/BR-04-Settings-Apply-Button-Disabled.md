# [Setting][UI] When the Player Opens the Settings Menu, 
the Apply Button is Not Clickable (Disabled)

## Description
When a player opens the game and changes the game 
settings in the settings menu, the apply button 
remains unclickable.

## Steps to Reproduce
1. Launch the game on the build 1.20.1
2. Open the setting menu
3. Change the game settings
4. On the bottom, click the Apply button 

## Expected Result
The game settings should be applied after they have been changed

## Actual Result
Even after changing settings, the game settings 
do not change and the apply button is not clickable

## Details
- **Platform:** PC, PS5, XBOX
- **Severity:** Medium
- **Reproducibility:** 5/5
- **Labels:** Setting, UI
