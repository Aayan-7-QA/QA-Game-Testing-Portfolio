# [UI][Input] Game Setting Button Becomes 
Unresponsive While Rapid Clicks "5-10 Times"

## Description
During gameplay, the user clicks the setting button 5-10 times, and
then the setting button becomes unresponsive after multiple rapid clicks
this situation causes only during gameplay while changing game settings
This causes users bad behavior, and sometimes settings don't change. 

## Steps to Reproduce
1. Launch the game on the build 1.20.309.209307
2. Start the gameplay
3. Click setting button quickly (5 to 10 times)
4. Observe the situation

## Expected Result
During gameplay, the setting button should become responsive and 
be fast after rapid clicks. 

## Actual Result
During gameplay, the settings button becomes unresponsive after rapid clicks

## Details
- **Platform:** PC
- **Severity:** Medium
- **Reproducibility:** 5/5
- **Labels:** UI, Input
