# [Game][Rapid Clicking] Character Behaves Incorrectly 
When Jump Button is Rapidly Clicked

## Description
When the player is playing the game and rapidly 
clicking the jump button, the character double 
jumps and gets stuck.

## Steps to Reproduce
1. Launch the game on the build 129.390.23478FG
2. Start the Gameplay/Mission
3. Rapidly click the jump button multiple times

## Expected Result
The game should work properly after clicking 
the jump button quickly and any additional 
clicks should be ignored

## Actual Result
The character performs multiple unintended jumps or enters a glitched state 
when the jump button is clicked rapidly

## Details
- **Platform:** PC
- **Severity:** Medium
- **Reproducibility:** 5/5
- **Labels:** Game, Rapid Clicking
