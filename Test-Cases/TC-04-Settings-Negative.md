# TC-04 — Settings Negative

| Field | Details |
|-------|---------|
| **ID** | QBGT-N-4 |
| **Summary** | The apply button stays inactive or grey when no settings are changed |
| **Pre-Condition** | - |
| **Status** | Pass |

## Steps
1. Boot the game on the build 1.0.20
2. Tap the setting menu
3. Click the apply button

## Actual Result
The apply button stays inactive and grey after no setting was changed in the menu

## Expected Result
The apply button remains inactive and grey after no settings are changed in the menu and the setting value remains the preferred input

## Post-Conditions
Settings menu is closed after clicked cross button
