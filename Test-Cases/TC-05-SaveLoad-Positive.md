# TC-05 — Save & Load Positive

| Field | Details |
|-------|---------|
| **ID** | QBGT-P-5 |
| **Summary** | Game progress is saved in the game server after 5 to 10 minutes of playing |
| **Pre-Condition** | The user is playing the game after it has been installed on the device |
| **Status** | Pass |

## Steps
1. Boot the game on the build 8.90.1
2. Start the gameplay/mission
3. Play for 5 to 10 minutes to make progress
4. Open the game menu
5. Click the Save button

## Actual Result
After making some progress, click the Save button the current game progress is saved on the game server

## Expected Result
When a player clicks the save icon/button after playing for a while, their game progress should be saved on the server
