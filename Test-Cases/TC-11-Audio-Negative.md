# TC-11 — Audio Negative

| Field | Details |
|-------|---------|
| **ID** | QBGT-N-11 |
| **Summary** | Gunshots sound are not audible after tapping the firing button during gameplay |
| **Pre-Condition** | The game is installed on the device and requires headphones |
| **Status** | Fail |

## Steps
1. Boot the game on the build 6789.1
2. Start the gameplay/mission
3. Tap the firing button

## Actual Result
During gameplay, after tapping the firing button, the gun sound is not audible, the gun-firing animation continues indefinitely, and no error message appears

## Expected Result
After tapping the firing button, the gun does not make a sound, and the animation continues, and an error message appears "Game audio not working"

## Post-Conditions
Game continues smoothly after firing
