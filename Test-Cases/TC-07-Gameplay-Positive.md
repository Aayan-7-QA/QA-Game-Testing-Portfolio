# TC-07 — Gameplay Positive

| Field | Details |
|-------|---------|
| **ID** | QBGT-P-7 |
| **Summary** | After tapping the character jump button, the character jumps and an animation is shown |
| **Pre-Condition** | The user is playing the game after it has been installed on the device |
| **Status** | Fail |

## Steps
1. Boot the game on the build 1.FKJ.209
2. Start the gameplay/mission
3. Tap the jump button

## Actual Result
While tapping the jump button, the character does not jump and no animation is displayed

## Expected Result
After tapping the jump button, the character should jump and animation should be displayed

## Post-Conditions
Character lands back on the ground after jumping
